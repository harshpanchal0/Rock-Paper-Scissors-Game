## Rock Paper Scissors 
 
**A basic Rock Paper Scissors game built using HTML/JavaScript/CSS. No dependencies.**

## Features

* Play Rock-Paper-Scissors against the computer.
* Random choice generation for the computer.
* Option to play multiple rounds.
